# SmartRoom-Security
SmartRoom Security je IoT projekt postavený na ESP32-S3, ktorý monitoruje pohyb a svetelné podmienky v miestnosti.
Pri detekcii pohybu v aktívnom režime spúšťa alarm.
Dáta sú zobrazované v prehľadnej webovej aplikácii s grafom svetla a vonkajšou teplotou získanou cez WEB API.

## Použité technológie
- ESP32-S3
- Arduino
- Wi-Fi
- WebServer
- Open-Meteo API
